# BOARDGAME_UNO
Ứng dụng Boardgame UNO được phát triển trong khuôn khổ đồ án môn học với mục tiêu mô phỏng lại trò chơi bài UNO nổi tiếng dưới dạng phần mềm chạy trên nền tảng Windows Forms (C#).
Dự án hướng đến các mục tiêu sau:

-Xây dựng phiên bản UNO cơ bản, cho phép người chơi tham gia trận đấu, thực hiện các hành động như: chia bài, đánh bài, rút bài, và xác định người thắng cuộc.
-Thiết kế giao diện người dùng trực quan, thân thiện, gồm các form chính:
  +Form Login
  +Form Lobby
  +Form Gameplay
  +Form Result
Áp dụng kiến thức lập trình hướng đối tượng (OOP) để xây dựng các lớp như: Card, Deck, Player, GameController, giúp quản lý logic trò chơi hiệu quả.
Thực hành quy trình phát triển phần mềm, bao gồm: phân tích yêu cầu, thiết kế, lập trình, kiểm thử, viết báo cáo, được thể hiện chi tiết trong file Timeline_boardgameUNO.xlsx.
Phát triển kỹ năng làm việc nhóm và sử dụng công cụ hỗ trợ quản lý tiến độ, phối hợp công việc giữa các thành viên.
